# 04 - HTML Lists

This folder contains separate files for each list type in HTML.

---

## 📂 Files Included

1. 01-unordered-list.html  
   → Unordered list using `<ul>`

2. 02-ordered-list.html  
   → Ordered list using `<ol>`

3. 03-type-attribute.html  
   → Ordered list with `type` attribute

4. 04-start-attribute.html  
   → Ordered list with `start` attribute

5. 05-nested-list.html  
   → Nested lists

6. 06-description-list.html  
   → Description list using `<dl>`, `<dt>`, `<dd>` (Extra but important)

---

## 🔹 Basic Tags
- `<ul>` → Unordered list
- `<ol>` → Ordered list
- `<li>` → List item
- `<dl>` → Description list
- `<dt>` → Description term
- `<dd>` → Description detail

## 🔹 Attributes & Advanced
- `type` → Numbering or bullets style
- `start` → Start number for ordered list
- Nested lists → Lists inside another list

---

Each file demonstrates one concept only, making it easy for beginners to learn step by step.
