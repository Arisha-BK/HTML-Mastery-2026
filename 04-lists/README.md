# 04 - HTML Lists

This folder contains separate files for each list type.

## Files Included

1. 01-unordered-list.html → <ul>
2. 02-ordered-list.html → <ol>
3. 03-type-attribute.html → type attribute
4. 04-start-attribute.html → start attribute
5. 05-nested-list.html → Nested lists
6. 06-description-list.html → <dl>, <dt>, <dd>

Each file demonstrates one concept only.
