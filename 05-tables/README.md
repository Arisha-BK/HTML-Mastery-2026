## 📘 05 - Tables

This section explains how to create tables in HTML using different table-related tags.  
Tables are used to organize data into rows and columns.

---

This folder covers all important table concepts.

## 🔹 Basic Tags
- `<table>`
- `<tr>`
- `<th>`
- `<td>`

## 🔹 Important Attributes
- `border`
- `cellpadding`
- `cellspacing`
- `colspan`
- `rowspan`

## 🔹 Advanced Structure
- `<thead>`
- `<tbody>`
- `<tfoot>`

Each file explains one concept clearly.
