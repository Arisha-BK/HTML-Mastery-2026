## 📘 08 - Media Tags

This section explains how to use audio, video, and iframe elements in HTML.  
These tags allow embedding media content into web pages.

---

## 📂 Files Included

1. 01-audio.html → Basic audio tag example
2. 02-video.html → Basic video tag example
3. 03-iframe.html → Basic iframe embedding example
4. 04-advanced-media.html → Advanced audio, video, and responsive iframe

---

## 🔹 Basic Media Tags
- `<audio>` → Embed audio files
- `<video>` → Embed video files
- `<iframe>` → Embed another webpage or content

---

## 🔹 Advanced Usage
- `controls` → Show player controls  
- `autoplay` → Start media automatically  
- `loop` → Repeat media  
- `muted` → Mute by default  
- `<source>` → Multiple formats support  
- Responsive iframes → Use `width="100%"`  

Each file explains one concept clearly from beginner to advanced usage.
