# 📘 02 - Text Formatting

This section explains how to format text in HTML using different tags.

---

## 🔹 Example Code

```html
<p><b>Bold text using b tag</b></p>
<p><strong>Bold text using strong tag</strong></p>

<p><i>Italic text using i tag</i></p>
<p><em>Italic text using em tag</em></p>

<p><mark>Highlighted text using mark tag</mark></p>
<p>This is <sub>subscript</sub> and <sup>superscript</sup></p>
<p><small>Smaller text using small tag</small></p>
