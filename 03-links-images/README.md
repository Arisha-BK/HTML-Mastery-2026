# 📘 03 - Links & Images

This section explains how to create hyperlinks and display images in HTML.

---

## 🔹 Anchor Tag (<a>)

### Example

```html
<a href="https://example.com">Visit Website</a>
<a href="https://example.com" target="_blank">Open in new tab</a>
<a href="mailto:someone@email.com">Send Email</a>
<a href="tel:+1234567890">Call Now</a>

⋆ <a> → Defines a hyperlink
⋆ href → Destination URL
⋆ target="_blank" → Opens link in new tab
⋆ mailto: → Creates email link
⋆ tel: → Creates phone link

---
🔹 Image Tag (<img>)
Example
<img src="image.jpg" alt="Description" width="200" height="200">


⋆ <img> → Displays an image
⋆ src → Image source path
⋆ alt → Alternative text (important for accessibility)
⋆ width & height → Control image size
