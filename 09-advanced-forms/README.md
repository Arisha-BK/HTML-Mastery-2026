## 📘 09 - Advanced Forms & Input Validation

This section covers advanced HTML form features and input validation.  
It helps create real-world forms with proper constraints and grouping.

---

## 📂 Files Included

1. 01-basic-validation.html → Required, pattern, minlength, maxlength
2. 02-email-url-number.html → Email, URL, Number inputs with min/max
3. 03-date-range-step.html → Date input, step, min, max
4. 04-fieldset-legend.html → Grouping fields with <fieldset> and <legend>
5. 05-combined-form.html → Full advanced form example combining all features

---

## 🔹 Key Features
- Required fields, pattern matching  
- Input types: email, url, number, date  
- Min, max, minlength, maxlength, step attributes  
- Fieldset & legend for grouping  
- Custom validation for user-friendly forms

Each file explains one concept clearly from beginner to advanced usage.
